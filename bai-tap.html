Bài tập tổng hợp

Tổng hợp bài tập từ cơ bản đến nâng cao, mô phỏng tình huống thực tế trong công việc. Mỗi bài có lời giải chi tiết.

🟢 Nhóm 1: Cơ bản (Chương 1-3)

📝 BT1 – Phân tích đơn hàng siêu thị

Tình huống: Bạn là nhân viên quản lý tại siêu thị MiniMart. Hãy phân tích dữ liệu đơn hàng tháng 1/2024.

Dữ liệu mẫu:

MaDHNgayNhomHangSanPhamSoLuongDonGiaThanhTien
MM00102/01/2024Thực phẩmGạo 5kg10120,0001,200,000
MM00202/01/2024Đồ uốngCoca Cola2410,000240,000
MM00303/01/2024Gia dụngNước rửa chén535,000175,000
... (tạo thêm 30+ dòng tương tự)

Yêu cầu:

  1. Chuyển dữ liệu thành Table, đặt tên tblDonHang
  2. Tạo PT: Doanh thu theo NhomHang
  3. Tạo PT: Doanh thu theo NhomHang và SanPham (2 fields trong Rows)
  4. Tạo PT: Doanh thu theo Ngày, group theo tuần
  5. Trả lời: Nhóm hàng nào bán chạy nhất? Ngày nào doanh thu cao nhất?

✅ Hướng dẫn:

Tạo Table (Ctrl+T), đặt tên tblDonHang trong Table Design

PT1: Insert → PivotTable → kéo NhomHang → Rows, ThanhTien → Values

PT2: Tạo PT mới → kéo NhomHang → Rows, SanPham → Rows (dưới NhomHang)

PT3: Kéo Ngay → Rows → click phải → Group → chọn Days, Number of days = 7

📝 BT2 – Báo cáo nhân sự công ty

Tình huống: Phòng HR cần báo cáo tổng quan nhân sự cho ban giám đốc.

Dữ liệu: MaNV, HoTen, PhongBan (5 phòng), ChucVu (NV/TL/TP/GD), GioiTinh, NgaySinh, NgayVao, LuongCoBan — 30+ nhân viên

Yêu cầu:

  1. PT: Số nhân viên theo PhongBan (Count)
  2. PT: Lương trung bình theo PhongBan × ChucVu (Rows: PhongBan+ChucVu, Values: Average of Luong)
  3. PT: Tỷ lệ giới tính theo PhongBan (Rows: PhongBan, Columns: GioiTinh, Values: Count of MaNV)

Tạo 3 PT riêng biệt. Lưu ý: để đếm nhân viên, kéo MaNV vào Values (tự động Count vì MaNV là Text). Để tính Average Luong: kéo LuongCoBan vào Values → click phải → Value Field Settings → chọn Average.

🟡 Nhóm 2: Trung bình (Chương 4-6)

📝 BT3 – Phân tích chi tiết doanh thu bán hàng

Tình huống: Giám đốc kinh doanh muốn biết chi tiết doanh thu, tỷ lệ đóng góp và xu hướng.

  1. Từ dữ liệu bán hàng, tạo PT: Rows = SanPham, Values = Sum of ThanhTien + % of Grand Total + Rank
  2. Sort theo doanh thu giảm dần
  3. Áp dụng Value Filter: Top 5
  4. Format: Style Medium, Number format #,##0 "₫", Tabular form
  5. Thêm Slicer cho KhuVuc + Timeline cho NgayBan

✅ Key points:

  • Kéo ThanhTien vào Values 3 lần → đổi lần 2 sang % of Grand Total, lần 3 sang Rank Largest to Smallest
  • Click phải cột giá trị → Sort → Largest to Smallest
  • Label dropdown → Value Filters → Top 10 → đổi thành Top 5 Items
  • PivotTable Analyze → Insert Slicer + Insert Timeline
📝 BT4 – So sánh doanh thu tháng trước vs tháng này

Yêu cầu:

  1. Tạo dữ liệu bán hàng trong 6 tháng (T1-T6/2024)
  2. PT: Rows = SanPham, Columns = NgayBan (group theo Month)
  3. Values: Sum of ThanhTien
  4. Thêm ThanhTien lần 2 → Show Values As: % Difference From Previous (Month)
  5. Trả lời: Sản phẩm nào tăng trưởng đều nhất? Sản phẩm nào giảm?

✅ Hướng dẫn Show Values As:

Kéo ThanhTien vào Values lần 2

Click phải → Value Field Settings → tab Show Values As

Chọn % Difference From → Base field: Ngay → Base item: (previous)

Cột mới hiện % thay đổi so với tháng trước. Giá trị dương = tăng, âm = giảm.

🔴 Nhóm 3: Nâng cao (Chương 7-10)

📝 BT5 – Power Pivot: Liên kết Đơn hàng - Sản phẩm - Khách hàng

Tình huống: Công ty có 3 bảng dữ liệu riêng biệt, cần tạo báo cáo tổng hợp.

  1. Tạo 3 Tables:
    • tblOrders: OrderID, CustomerID, ProductID, Qty, OrderDate (30 dòng)
    • tblProducts: ProductID, Name, Category, Price (8 sản phẩm)
    • tblCustomers: CustomerID, Name, City, Region (10 khách)
  2. Tạo Relationships giữa 3 bảng
  3. Tạo PT từ Data Model: Rows = Product Name + Category, Columns = City, Values = Sum of Qty
  4. Bonus DAX: Tạo Measure Revenue := SUMX(tblOrders, tblOrders[Qty] * RELATED(tblProducts[Price]))

✅ Hướng dẫn:

Tạo 3 Tables trên 3 sheets → đặt tên phù hợp

Data → Relationships → New → tạo 2 relationships (Orders→Products, Orders→Customers)

Insert → PivotTable → tick "Add this data to the Data Model"

Trong PivotTable Fields → chuyển sang "ALL" → expand từng bảng → kéo fields

Mở Power Pivot window → Measures → New Measure → nhập công thức DAX

📝 BT6 – Power Query: Gộp 12 file báo cáo tháng

Tình huống: Mỗi chi nhánh gửi báo cáo Excel hàng tháng. Bạn cần gộp 12 file lại.

  1. Tạo 3 file Excel giả lập (BaoCaoT1.xlsx, BaoCaoT2.xlsx, BaoCaoT3.xlsx) — cùng cấu trúc: Ngay, SanPham, SoLuong, DoanhThu
  2. Đặt cả 3 file vào 1 folder
  3. Dùng Power Query: Data → From Folder → Combine & Transform
  4. Trong PQ Editor: đổi kiểu dữ liệu, xóa cột không cần, rename cột Source.Name thành "ThangBaoCao"
  5. Close & Load → Tạo Pivot Table từ dữ liệu gộp

✅ Key steps:

  • From Folder → Combine & Transform Data → chọn Sheet1 trong mỗi file
  • PQ Editor: đổi kiểu Ngay sang Date, SoLuong sang Whole Number
  • Click phải cột Source.Name → Rename → "ThangBaoCao"
  • Có thể dùng Transform → Extract → Text Before Delimiter (".") để bỏ đuôi .xlsx
  • Close & Load → Insert PivotTable → phân tích theo tháng
🏆 BT7 – Dự án cuối: Sales Dashboard hoàn chỉnh

🏆 Đây là bài tập TỔNG HỢP — thể hiện toàn bộ kiến thức!

Yêu cầu: Xây dựng Sales Dashboard hoàn chỉnh với:

  1. Dữ liệu: 100+ đơn hàng, 6 tháng, 3 khu vực, 10 sản phẩm, 6 nhân viên
  2. KPI Cards: Tổng doanh thu, Số đơn hàng, Doanh thu TB/đơn, Tăng trưởng %
  3. Charts:
    • Line: Doanh thu theo tháng
    • Bar: Top 5 sản phẩm
    • Doughnut: Tỷ lệ doanh thu theo khu vực
    • Column: Doanh thu theo nhóm sản phẩm
  4. Table: Top 5 nhân viên (bảng formatted)
  5. Interactive: 2 Slicer (KhuVuc, NhomSP) + 1 Timeline → kết nối tất cả PT
  6. Hoàn thiện: Ẩn gridlines, ẩn sheets data, format chuyên nghiệp
