Pivot Chart

Biến Pivot Table thành biểu đồ trực quan. Tìm hiểu các loại chart phù hợp, cách tùy chỉnh và kết hợp với Slicer.

📊 7.1 Tạo Pivot Chart

Pivot Chart là biểu đồ liên kết trực tiếp với Pivot Table. Khi PT thay đổi, chart tự cập nhật.

Cách 1: Từ Pivot Table có sẵn

Click vào Pivot Table → tab PivotTable Analyze

Click PivotChart

Chọn loại chart (Column, Bar, Line, Pie...) → OK

Cách 2: Tạo PivotChart trực tiếp

Chọn dữ liệu nguồn → tab Insert

Click PivotChart (trong nhóm Charts)

Excel tạo cả Pivot Table + Pivot Chart cùng lúc

📌
Ghi nhớ: Pivot Chart luôn đi kèm 1 Pivot Table. Xóa PT → chart cũng bị xóa. Lọc trên PT → chart tự lọc theo.

📋 7.2 Chọn loại Chart phù hợp

📊 Hướng dẫn chọn Chart

Mục đíchLoại ChartVí dụ
So sánh giá trịColumn / BarDoanh thu theo sản phẩm
Tỷ lệ phần trămPie / Doughnut% doanh thu mỗi khu vực
Xu hướng theo thời gianLine / AreaDoanh thu theo tháng
So sánh + xu hướngCombo (Column + Line)Doanh thu (cột) + tỷ lệ tăng trưởng (đường)
Phân tán / tương quanScatterSố lượng vs Doanh thu
So sánh nhiều nhómStacked Column/BarDoanh thu theo vùng, chia theo sản phẩm
💡
Nguyên tắc vàng: 1 chart chỉ nên truyền tải 1 thông điệp. Không nhồi nhét quá nhiều dữ liệu vào 1 biểu đồ.

🎨 7.3 Tùy chỉnh Pivot Chart

🔧 Các tùy chỉnh quan trọng

  • Chart Title: Click tiêu đề → gõ tên descriptive (ví dụ: "Doanh thu Q1/2024 theo sản phẩm")
  • Data Labels: Hiển thị giá trị trên cột/điểm → Chart Design → Add Chart Element → Data Labels
  • Legend: Chú thích → có thể đặt Top, Bottom, Left, Right
  • Axis: Đổi min/max, format số, ẩn trục → Click phải vào trục → Format Axis
  • Colors: Click phải vào chuỗi → Format Data Series → Fill → chọn màu
  • Chart Styles: Tab Design → Chart Styles → chọn style có sẵn

🔲 Ẩn Field Buttons

Pivot Chart mặc định hiện các dropdown button trên chart. Để ẩn:

  • Tab PivotChart Analyze → Field Buttons → chọn Hide All
  • Hoặc click phải vào button → Hide All Field Buttons on Chart

→ Chart sẽ sạch sẽ hơn, phù hợp trình bày và Dashboard.

🔗 7.4 Kết hợp Pivot Chart + Slicer

Khi Slicer kết nối với Pivot Table, Pivot Chart tự động cập nhật theo:

Tạo Pivot Table + Pivot Chart

Insert Slicer (từ Chương 6)

Click nút trên Slicer → Chart thay đổi ngay lập tức

💡
Mẹo Dashboard: Ẩn Pivot Table (thu nhỏ hàng), chỉ hiện Pivot Chart + Slicer. Giám đốc click Slicer → chart thay đổi → trải nghiệm như BI tool!

🎯 7.5 Bài tập thực hành

📝 Bài tập 7.1 – Biểu đồ doanh thu theo sản phẩm

Yêu cầu:

  1. Tạo Pivot Table: Rows = SanPham, Values = Sum of ThanhTien
  2. Sort doanh thu từ cao → thấp
  3. Tạo Pivot Chart dạng Bar (thanh ngang)
  4. Thêm Data Labels hiển thị giá trị
  5. Đổi title thành "Top Sản phẩm theo Doanh thu"
  6. Ẩn Field Buttons
  7. Thêm Slicer KhuVuc → test lọc

✅ Hướng dẫn:

Tạo PT → Sort → PivotTable Analyze → PivotChart → chọn Bar

Click chart → Chart Design → Add Chart Element → Data Labels → Outside End

Click title → gõ "Top Sản phẩm theo Doanh thu"

PivotChart Analyze → Field Buttons → Hide All

PivotTable Analyze → Insert Slicer → tick KhuVuc → OK

📝 Bài tập 7.2 – Biểu đồ xu hướng doanh thu theo tháng

Yêu cầu:

  1. Tạo PT: Rows = NgayBan (group theo Month), Values = Sum of ThanhTien
  2. Tạo Pivot Chart dạng Line with Markers
  3. Thêm Data Labels + tùy chỉnh màu đường xanh lá
  4. Thêm trendline (click phải vào đường → Add Trendline → Linear)
  5. Trả lời: Xu hướng doanh thu tăng hay giảm?

✅ Hướng dẫn:

Group NgayBan theo Months → Insert PivotChart → Line with Markers. Trendline cho thấy xu hướng chung (tăng/giảm). Đường trendline đi lên = doanh thu tăng.
