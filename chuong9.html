Power Query kết hợp Pivot Table

Tự động hóa việc thu thập, làm sạch và kết hợp dữ liệu từ nhiều nguồn trước khi tạo Pivot Table.

🔄 9.1 Power Query là gì?

Power Query (còn gọi là "Get & Transform") là công cụ ETL (Extract, Transform, Load) tích hợp sẵn trong Excel 2016+.

📊 Quy trình ETL

BướcTênHoạt động
EExtract (Trích xuất)Lấy dữ liệu từ CSV, Excel, Web, Database, API...
TTransform (Biến đổi)Lọc, đổi kiểu dữ liệu, tách/gộp cột, pivot/unpivot...
LLoad (Nạp)Đưa vào Excel Table hoặc Data Model → sẵn sàng cho Pivot Table
💡
Sức mạnh của Power Query: Sau khi thiết lập 1 lần, chỉ cần nhấn Refresh → Power Query tự động lấy dữ liệu mới, làm sạch, và cập nhật Pivot Table. Tiết kiệm hàng giờ công việc!

📥 9.2 Import dữ liệu

Power Query hỗ trợ import dữ liệu từ hầu hết mọi nguồn:

📋 Các nguồn dữ liệu phổ biến

NguồnCách mởỨng dụng
File CSV/TextData → From Text/CSVExport từ ERP, POS, hệ thống cũ
File Excel khácData → From WorkbookTổng hợp báo cáo chi nhánh
FolderData → From FolderGộp nhiều file cùng cấu trúc
WebData → From WebCrawl bảng giá, tỷ giá...
SQL ServerData → From SQL ServerKết nối database trực tiếp
Bảng trong fileData → From Table/RangeBiến đổi dữ liệu đã có

Vào tab Data → nhóm Get & Transform Data

Chọn nguồn dữ liệu phù hợp

Power Query Editor mở ra → dữ liệu được preview

🔧 9.3 Các thao tác Transform phổ biến

📋 Thao tác Transform hay dùng

Thao tácCách làmVí dụ

Xóa cộtClick phải cột → Remove ColumnsXóa cột không cần thiết
Lọc dòngClick mũi tên ▼ trên headerLọc bỏ dòng trống, dòng lỗi

📁 9.4 Gộp nhiều file từ Folder

Tình huống phổ biến: bạn có 12 file báo cáo tháng (cùng cấu trúc) cần gộp lại:

Đặt tất cả file vào 1 folder

Data → From Folder → chọn folder → OK

Click Combine & Transform Data

Chọn sheet/table trong mỗi file → OK

Power Query tự gộp tất cả file + thêm cột Source.Name (tên file gốc)

Close & Load → dữ liệu gộp sẵn sàng cho Pivot Table

💡
Tự động hóa: Khi thêm file mới vào folder → chỉ cần Refresh All (Ctrl + Alt + F5) → Power Query tự phát hiện file mới và gộp vào!

🔗 9.5 Merge & Append Queries

🔗 Merge Queries (Nối ngang — giống VLOOKUP)

Kết nối 2 bảng theo cột chung:

  • Home → Merge Queries
  • Chọn bảng thứ 2 + cột khóa → chọn kiểu Join (Left, Inner, Full...)
  • Expand cột kết quả để lấy thông tin từ bảng 2

📋 Append Queries (Nối dọc — gộp dữ liệu)

Xếp chồng dữ liệu từ nhiều bảng cùng cấu trúc:

  • Home → Append Queries
  • Chọn "Two tables" hoặc "Three or more tables"
  • Kết quả: 1 bảng gộp tất cả dòng

🎯 9.6 Bài tập thực hành

📝 Bài tập 9.1 – Import & Transform CSV

Tình huống: Nhận file báo cáo bán hàng dạng CSV từ hệ thống POS. Dữ liệu lộn xộn, cần làm sạch.

  1. Tạo file CSV mẫu với 20 dòng (có ô trống, "N/A", ngày dạng text)
  2. Import vào Excel bằng Power Query (Data → From Text/CSV)
  3. Trong Power Query Editor:
    • Đổi kiểu cột ngày từ Text → Date
    • Replace "N/A" thành "0" ở cột số
    • Xóa dòng trống (Remove Rows → Remove Blank Rows)
    • Thêm cột ThanhTien = SoLuong × DonGia
  4. Close & Load → Tạo Pivot Table từ kết quả

✅ Các bước trong Power Query Editor:

Data → From Text/CSV → chọn file → Load/Transform Data

Click icon kiểu cột NgayBan → đổi thành Date

Click phải cột SoLuong → Replace Values: "N/A" → "0"

Đổi kiểu SoLuong sang Whole Number

Home → Remove Rows → Remove Blank Rows

Add Column → Custom Column: = [SoLuong] * [DonGia]

Home → Close & Load → Insert → PivotTable
