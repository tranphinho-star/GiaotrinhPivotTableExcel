Giới thiệu Pivot Table

Tìm hiểu Pivot Table là gì, tại sao nó là công cụ phân tích dữ liệu mạnh nhất trong Excel, và cách chuẩn bị dữ liệu đúng cách.

📖 1.1 Pivot Table là gì?

Pivot Table (Bảng tổng hợp) là công cụ phân tích dữ liệu mạnh mẽ nhất trong Microsoft Excel. Nó cho phép bạn tóm tắt, phân tích, khám phá và trình bày dữ liệu lớn chỉ bằng vài thao tác kéo thả — không cần viết công thức phức tạp.

🔄 Tại sao gọi là "Pivot"?

Từ "Pivot" nghĩa là "xoay". Pivot Table cho phép bạn xoay dữ liệu theo nhiều góc nhìn khác nhau: theo thời gian, theo sản phẩm, theo khu vực, theo nhân viên... chỉ bằng cách kéo thả các trường dữ liệu.

💡 Ví dụ dễ hiểu

Bạn có bảng dữ liệu 10,000 đơn hàng. Thay vì dùng SUMIFS, COUNTIFS để tạo báo cáo, Pivot Table giúp bạn:

  • Tổng doanh thu theo từng sản phẩm → kéo "Sản phẩm" vào Rows, "Doanh thu" vào Values
  • So sánh theo quý → kéo "Ngày" vào Columns (tự động group theo quý)
  • Lọc theo khu vực → kéo "Khu vực" vào Filters

Tất cả chỉ trong 30 giây, không cần viết bất kỳ công thức nào!

⚠️ 1.3 Chuẩn bị dữ liệu đúng cách

Trước khi tạo Pivot Table, dữ liệu nguồn phải được chuẩn bị đúng cách. Đây là bước quan trọng nhất mà nhiều người bỏ qua.

✅ 6 Quy tắc vàng cho dữ liệu nguồn

  1. Mỗi cột là một trường dữ liệu — Tên cột rõ ràng, không trùng lặp
  2. Mỗi hàng là một bản ghi — Không để hàng trống xen giữa
  3. Không merge cells — Pivot Table không đọc được ô gộp
  4. Dữ liệu nhất quán — Cùng một cột phải cùng kiểu dữ liệu (số/text/ngày)
  5. Có dòng tiêu đề — Hàng đầu tiên phải là tên cột
  6. Không có hàng tổng ở cuối — Pivot Table tự tính tổng

📐 1.4 Chuyển dữ liệu thành Table

Bước quan trọng trước khi tạo Pivot Table là chuyển dữ liệu sang dạng Table:

Chọn bất kỳ ô nào trong vùng dữ liệu

Nhấn Ctrl + T hoặc vào Insert → Table

Excel tự động nhận diện vùng dữ liệu → Đánh dấu "My table has headers"

Nhấn OK → Dữ liệu được chuyển thành Table với style mặc định

Đặt tên Table ý nghĩa trong Table Design → Table Name (ví dụ: tblDonHang)

🎯 Lợi ích của Table

  • Tự động mở rộng: Thêm dòng mới → Table tự thêm vào
  • Structured References: Công thức dễ đọc hơn ([@Cột] thay vì A2)
  • Pivot Table cập nhật: Khi Refresh, Pivot Table tự lấy dữ liệu mới
  • Filter & Sort: Table có sẵn nút filter ở header

🎯 1.6 Bài tập thực hành

📝 Bài tập 1.1 – Chuẩn bị dữ liệu bán hàng

Yêu cầu:

  1. Tạo file Excel mới, đặt tên "DuLieuBanHang.xlsx"
  2. Tạo bảng dữ liệu với các cột: MaDH, NgayBan, KhuVuc, NhanVien, SanPham, SoLuong, DonGia, ThanhTien
  3. Nhập ít nhất 20 dòng dữ liệu mẫu (3 khu vực, 5 sản phẩm, 4 nhân viên)
  4. Chuyển dữ liệu thành Table (Ctrl + T), đặt tên tblBanHang
  5. Kiểm tra: không có ô trống, không merge cells, cột ThanhTien = SoLuong × DonGia

Yêu cầu: Xem bảng dữ liệu sau và chỉ ra tất cả các lỗi khiến Pivot Table không hoạt động đúng:

✅ Các lỗi tìm thấy:

  1. Dạng crosstab: Tháng 1, 2, 3 nên là giá trị trong cột "Tháng", không phải tên cột
  2. Merged cells: Hàng 3 có ô gộp → Pivot Table không đọc được
  3. Ô trống: Tháng 3 của TV Samsung bị trống → nên điền 0
  4. Dữ liệu không nhất quán: "Không có" trong cột số → nên là 0
  5. Hàng tổng: Hàng cuối là tổng → Pivot Table sẽ tính tổng sai
