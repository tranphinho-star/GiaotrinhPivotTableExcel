Lọc, Sắp xếp & Slicer

Làm chủ các công cụ lọc dữ liệu: Report Filter, Label/Value Filter, Sort, Slicer và Timeline.

🔲 6.1 Report Filter

Report Filter là bộ lọc ở phía trên Pivot Table, lọc toàn bộ bảng theo 1 điều kiện.

Kéo field (ví dụ: KhuVuc) vào vùng Filters

Dropdown xuất hiện phía trên PT → Click để chọn giá trị

Tick "Select Multiple Items" nếu muốn chọn nhiều giá trị

💡
Mẹo: Report Filter hiển thị "(All)" khi không lọc, và "(Multiple Items)" khi chọn nhiều giá trị. Nhìn vào đây để biết đang lọc gì.

🏷️ 6.2 Label Filter & Value Filter

Lọc trực tiếp trên Row Labels hoặc Column Labels:

🏷️ Label Filter (Lọc theo nhãn)

Click mũi tên ▼ cạnh Row Labels → Label Filters:

Điều kiệnVí dụ
Equals / Does Not EqualSanPham = "Laptop"
Begins With / Ends WithBắt đầu bằng "Đ" → Điện thoại
Contains / Does Not ContainChứa "Tab" → Tablet
BetweenGiữa "A" và "L" (theo alphabet)

📊 Value Filter (Lọc theo giá trị)

Click mũi tên ▼ → Value Filters:

Điều kiệnVí dụ
Greater ThanDoanh thu > 10,000,000
Less Than or EqualDoanh thu ≤ 50,000,000
BetweenDoanh thu từ 10M đến 50M
Top 10Top 5 sản phẩm doanh thu cao nhất
📌
Top 10 Filter rất hữu ích: chọn Top/Bottom + số lượng + Items/Percent/Sum. Ví dụ: "Top 3 Items by Sum of ThanhTien" → chỉ hiện 3 sản phẩm bán chạy nhất.

🔀 6.3 Sắp xếp (Sort)

📋 Các cách Sort trong Pivot Table

CáchThao tácKết quả
Sort A→Z / Z→AClick phải → Sort → chọn hướngSắp xếp theo nhãn (alphabet)
Sort by ValueClick phải vào cột Values → Sort → Largest to SmallestSắp xếp theo doanh thu cao→thấp
Manual SortKéo thả item lên/xuốngSắp xếp tùy ý
More Sort OptionsClick phải → More Sort Options...Cấu hình nâng cao
💡
Mẹo: Sort by Value (lớn→nhỏ) giúp bạn nhanh chóng thấy item quan trọng nhất lên đầu. Rất hữu ích khi trình bày cho sếp!

🎛️ 6.4 Slicer

Slicer là bộ lọc trực quan dạng nút bấm, giúp lọc dữ liệu nhanh và đẹp hơn Report Filter.

Click vào Pivot Table → tab PivotTable Analyze

Click Insert Slicer

Tick các field muốn tạo Slicer (ví dụ: KhuVuc, SanPham)

Nhấn OK → Các Slicer panel xuất hiện

Click vào nút để lọc. Giữ Ctrl để chọn nhiều giá trị

🎨 Tùy chỉnh Slicer

  • Style: Click Slicer → tab Slicer → chọn style màu sắc
  • Columns: Slicer Settings → Number of columns (hiển thị nút theo nhiều cột)
  • Size: Kéo góc để resize
  • Kết nối nhiều PT: Click phải Slicer → Report Connections... → tick các PT muốn kết nối
🔥
Siêu mạnh: 1 Slicer có thể điều khiển nhiều Pivot Table cùng lúc → Đây là nền tảng để tạo Dashboard (Chương 10).

📅 6.5 Timeline

Timeline là Slicer đặc biệt cho dữ liệu ngày tháng — dạng thanh trượt thời gian.

Tab PivotTable Analyze → Insert Timeline

Chọn field ngày (NgayBan) → OK

Chọn cấp độ thời gian: Years, Quarters, Months, Days

Click hoặc kéo thanh trượt để lọc theo khoảng thời gian

💡
Mẹo: Timeline rất trực quan trên Dashboard. Người dùng chỉ cần kéo thanh trượt để xem dữ liệu từng tháng/quý/năm mà không cần hiểu Pivot Table.

🎯 6.6 Bài tập thực hành

📝 Bài tập 6.1 – Slicer theo vùng miền

Tình huống: Giám đốc muốn xem nhanh doanh thu theo khu vực bằng cách click nút.

Yêu cầu:

  1. Tạo Pivot Table: Rows = SanPham, Values = Sum of ThanhTien
  2. Insert Slicer cho KhuVuc
  3. Đổi style Slicer sang màu xanh lá (phù hợp báo cáo)
  4. Thiết lập Slicer hiển thị 3 cột
  5. Test: Click "Bắc" → xem PT thay đổi, Click "Nam" → xem PT thay đổi
  6. Thêm Timeline cho NgayBan → lọc theo tháng 1/2024

✅ Hướng dẫn:

Tạo PT → Kéo SanPham → Rows, ThanhTien → Values

PivotTable Analyze → Insert Slicer → tick KhuVuc → OK

Click Slicer → Slicer tab → chọn style xanh lá

Slicer tab → Columns: nhập 3

PivotTable Analyze → Insert Timeline → tick NgayBan → OK

Trên Timeline, chọn cấp "Months" → click vào Jan 2024

📝 Bài tập 6.2 – Top 3 sản phẩm bán chạy

Yêu cầu:

  1. Tạo PT: Rows = SanPham, Values = Sum of ThanhTien
  2. Sort: Sắp xếp theo doanh thu từ cao xuống thấp
  3. Áp dụng Value Filter → Top 10: Top 3 Items
  4. Kết quả: chỉ hiện 3 sản phẩm có doanh thu cao nhất

✅ Hướng dẫn:

Tạo PT bình thường với SanPham ở Rows

Click phải vào cột doanh thu → Sort → Largest to Smallest

Click mũi tên ▼ bên Row Labels → Value Filters → Top 10...

Đổi thành: Top 3 Items by Sum of ThanhTien → OK

Chỉ còn 3 sản phẩm: Điện thoại, Laptop, Tablet (theo dữ liệu mẫu).
