Tạo Pivot Table đầu tiên

Hướng dẫn từng bước tạo Pivot Table từ A đến Z, làm quen với giao diện PivotTable Fields.

🚀 2.1 Các bước tạo Pivot Table

Giả sử bạn đã có bảng dữ liệu bán hàng đã chuẩn bị ở Chương 1. Hãy làm theo các bước sau:

Chọn vùng dữ liệu: Click vào bất kỳ ô nào trong bảng dữ liệu (hoặc Table)

Mở lệnh: Vào tab Insert → Click PivotTable (hoặc Alt + N + V)

Chọn nguồn: Excel tự nhận diện vùng dữ liệu/Table → Kiểm tra lại cho đúng

Chọn vị trí: New Worksheet (sheet mới) hoặc Existing Worksheet (sheet hiện tại)

Nhấn OK: Pivot Table trống xuất hiện kèm panel PivotTable Fields bên phải

📌
Lưu ý: Nếu dữ liệu nguồn là Table (Ctrl+T), Excel sẽ hiển thị tên Table thay vì vùng ô (ví dụ: tblBanHang). Đây là cách tốt nhất vì Table tự động mở rộng.

🖥️ 2.2 Giao diện PivotTable Fields

Sau khi tạo, panel PivotTable Fields xuất hiện ở bên phải màn hình. Đây là "bộ não" của Pivot Table.

📋 Cấu trúc panel PivotTable Fields

Panel gồm 2 phần chính:

  1. Danh sách Fields (phía trên): Tất cả các cột trong dữ liệu nguồn hiển thị dưới dạng checkbox
  2. 4 vùng kéo thả (phía dưới):
    • 🔲 Filters — Lọc toàn bộ Pivot Table
    • 📏 Columns — Tạo tiêu đề cột
    • 📐 Rows — Tạo nhãn hàng
    • 📊 Values — Chứa giá trị tính toán (Sum, Count...)

🎯 Cách kéo thả Fields

Thao tácCách làm
Thêm field vào vùngKéo thả field từ danh sách vào vùng mong muốn, hoặc ✅ tick checkbox
Xóa field khỏi vùngKéo field ra khỏi vùng, hoặc bỏ tick ☐ checkbox
Di chuyển fieldKéo thả field từ vùng này sang vùng khác
Thay đổi thứ tựKéo field lên/xuống trong cùng một vùng
💡
Mẹo: Khi tick checkbox, Excel tự quyết định vùng: trường Text → Rows, trường Number → Values (Sum). Nhưng tốt nhất bạn nên kéo thả thủ công để kiểm soát hoàn toàn.

🔄 2.4 Refresh & Cập nhật dữ liệu

Khi dữ liệu nguồn thay đổi (thêm, sửa, xóa), Pivot Table không tự cập nhật. Bạn cần Refresh:

🔄 Các cách Refresh Pivot Table

CáchThao tácPhạm vi
Cách 1Click phải vào Pivot Table → RefreshPT hiện tại
Cách 2Alt + F5PT hiện tại
Cách 3Ctrl + Alt + F5Tất cả PT trong workbook
Cách 4Tab PivotTable Analyze → RefreshPT hiện tại hoặc All
⚠️
Quan trọng: Nếu dữ liệu nguồn là Range (không phải Table), khi thêm dòng mới ngoài range cũ, bạn phải vào PivotTable Analyze → Change Data Source để mở rộng vùng dữ liệu. Dùng Table sẽ tránh được vấn đề này!

🎯 2.6 Bài tập thực hành

📝 Bài tập 2.1 – Tạo Pivot Table đầu tiên

Yêu cầu:

  1. Mở file DuLieuBanHang.xlsx từ Bài tập 1.1
  2. Tạo Pivot Table ở sheet mới
  3. Tạo báo cáo "Tổng doanh thu theo khu vực" (KhuVuc → Rows, ThanhTien → Values)
  4. Thêm dòng dữ liệu mới vào bảng nguồn, rồi Refresh Pivot Table
  5. Khu vực nào có doanh thu cao nhất?

✅ Hướng dẫn:

Mở file → Click ô bất kỳ trong Table

Insert → PivotTable → chọn New Worksheet → OK

Kéo KhuVuc vào Rows

Kéo ThanhTien vào Values

Quay lại sheet dữ liệu → Thêm 1 dòng mới

Click trở lại Pivot Table → Alt + F5 (Refresh)

Pivot Table sẽ tự cập nhật với dòng mới. Khu vực có doanh thu cao nhất sẽ hiển thị rõ ở bảng kết quả.

📝 Bài tập 2.2 – Pivot Table nhiều fields

Yêu cầu:

  1. Tạo Pivot Table mới từ cùng bảng dữ liệu
  2. Kéo KhuVuc vào Rows, SanPham cũng vào Rows (dưới KhuVuc)
  3. Kéo ThanhTien vào Values
  4. Kéo SoLuong cũng vào Values
  5. Quan sát: Pivot Table hiển thị cả tổng tiền và tổng số lượng theo khu vực → sản phẩm
