Chương 04

Định dạng & Tùy chỉnh

Làm cho Pivot Table đẹp mắt, chuyên nghiệp và dễ đọc hơn với Styles, Report Layout, Subtotals và Number Format.

🎨 4.1 PivotTable Styles

Excel cung cấp hàng chục style có sẵn để thay đổi giao diện Pivot Table chỉ bằng 1 click:

Click vào Pivot Table → chuyển sang tab Design

Trong nhóm PivotTable Styles, hover qua các style để xem preview

Click chọn style phù hợp (Light, Medium, Dark)

📋 Các nhóm Style

NhómĐặc điểmKhi nào dùng
LightNền sáng, viền nhẹIn ấn, báo cáo chính thức
MediumCó màu nền header, xen kẽ màu hàngTrình bày, slide, dashboard
DarkNền tối, chữ sángMàn hình lớn, dashboard tối

✅ Tùy chọn bổ sung trong Design tab

  • Row Headers: Đậm hàng tiêu đề
  • Column Headers: Đậm cột tiêu đề
  • Banded Rows: Xen kẽ màu hàng (dễ đọc)
  • Banded Columns: Xen kẽ màu cột

📐 4.2 Report Layout

Report Layout quyết định cách sắp xếp dữ liệu trong Pivot Table. Có 3 dạng:

📦 Compact Form (Mặc định)

Tất cả Row fields gộp vào 1 cột duy nhất, với thụt lề phân cấp. Tiết kiệm không gian, phù hợp khi có nhiều field.

Row LabelsSum of ThanhTien
▼ Bắc50,000,000
   Laptop45,000,000
   Tai nghe5,000,000

📋 Outline Form

Mỗi Row field có cột riêng, nhãn nhóm ở đầu nhóm. Rõ ràng hơn, dễ đọc.

KhuVucSanPhamSum of ThanhTien
Bắc50,000,000
Laptop45,000,000
Tai nghe5,000,000

📊 Tabular Form

Mỗi field 1 cột riêng, nhãn nhóm lặp lại ở mỗi hàng. Giống bảng dữ liệu thông thường, phù hợp để xuất ra CSV/copy.

KhuVucSanPhamSum of ThanhTien
BắcLaptop45,000,000
BắcTai nghe5,000,000
NamChuột4,500,000
NamĐiện thoại64,000,000

Click Pivot Table → tab Design

Click Report Layout → chọn Compact/Outline/Tabular

Trong Tabular Form, có thể chọn thêm "Repeat All Item Labels"

💡
Mẹo thực tế: Dùng Tabular Form + Repeat All Item Labels khi cần copy dữ liệu từ Pivot Table vào bảng mới hoặc xuất CSV — vì mỗi hàng sẽ có đầy đủ thông tin.

🔢 4.3 Number Format

Mặc định, số trong Pivot Table không có định dạng. Hãy format cho chuyên nghiệp:

Click chuột phải vào ô số bất kỳ trong Values

Chọn Number Format... (hoặc Value Field Settings → Number Format)

Chọn Number → Decimal places: 0 → Tick ✅ "Use 1000 Separator"

Nhấn OK → Tất cả ô values được format đồng loạt

📊 Các format phổ biến

LoạiFormat codeVí dụ
Số nguyên + dấu phẩy#,##01,234,567
Tiền VNĐ#,##0 "₫"1,234,567 ₫
Phần trăm0.0%45.5%
Triệu đồng#,##0,, "Tr"1 Tr
Số âm đỏ#,##0;[Red]-#,##0-1,234

📏 4.4 Subtotals & Grand Totals

📐 Subtotals (Tổng phụ)

  • Bật/tắt: Design tab → Subtotals → Show/Don't Show
  • Vị trí: Show at Top of Group / Show at Bottom of Group
  • Subtotals tự hiện khi có nhiều Row fields (tổng cho mỗi nhóm chính)

📊 Grand Totals (Tổng cộng)

  • 4 tùy chọn:
  • Off for Rows and Columns — Tắt tất cả
  • On for Rows and Columns — Bật tất cả
  • On for Rows Only — Chỉ tổng hàng (cột Grand Total bên phải)
  • On for Columns Only — Chỉ tổng cột (hàng Grand Total bên dưới)
📌
Ghi nhớ: Khi làm Dashboard, thường tắt Grand Totals vì biểu đồ sẽ tự hiển thị tổng. Khi in báo cáo, nên bật Subtotals + Grand Totals.

🎯 4.5 Bài tập thực hành

📝 Bài tập 4.1 – Tạo báo cáo nhân sự đẹp mắt

Tình huống: Bạn là trợ lý HR, cần tạo báo cáo nhân sự cho sếp.

Yêu cầu:

  1. Tạo bảng dữ liệu nhân sự: MaNV, HoTen, PhongBan, ChucVu, GioiTinh, Luong (15+ dòng)
  2. Tạo Pivot Table: Rows = PhongBan + ChucVu, Values = Count of MaNV + Average of Luong
  3. Áp dụng style Medium 14 (xanh lá)
  4. Chuyển sang Outline Form + Repeat All Item Labels
  5. Format lương: #,##0 "₫"
  6. Bật Subtotals ở Bottom, Grand Totals cho cả Rows & Columns

✅ Hướng dẫn:

Tạo Table nhân sự → Ctrl + T → đặt tên tblNhanSu

Insert → PivotTable → New Worksheet

Kéo PhongBan → Rows, sau đó kéo ChucVu xuống dưới PhongBan trong Rows

Kéo MaNV → Values (tự động Count), kéo Luong → Values (chuyển sang Average)

Design → Styles → chọn Medium 14

Design → Report Layout → Show in Outline Form

Design → Report Layout → Repeat All Item Labels

Click phải vào ô lương → Number Format → #,##0 "₫"
