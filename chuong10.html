Dashboard thực tế

Kết hợp Pivot Table + Pivot Chart + Slicer + Timeline để tạo Dashboard chuyên nghiệp như BI tool.

📋 10.1 Dashboard là gì?

Dashboard (Bảng điều khiển) là trang tổng hợp hiển thị các chỉ số quan trọng (KPIs) dưới dạng biểu đồ và bảng, giúp người dùng nắm bắt tình hình chỉ trong vài giây.

🎯 Nguyên tắc thiết kế Dashboard

  1. Ít hơn là nhiều hơn: Tối đa 6-8 biểu đồ/bảng
  2. KPIs nổi bật: Các chỉ số chính (doanh thu, % tăng trưởng) ở vị trí dễ thấy nhất
  3. Nhất quán: Cùng font, cùng màu, cùng style xuyên suốt
  4. Interactive: Slicer + Timeline để người dùng tự khám phá
  5. 1 trang: Toàn bộ Dashboard trên 1 sheet, không cần cuộn

🏗️ 10.2 Các bước xây dựng Dashboard

Chuẩn bị dữ liệu: Dùng Power Query (Chương 9) để import, làm sạch, kết hợp dữ liệu

Tạo Data Model: Thiết lập Relationships giữa các bảng (Chương 8)

Tạo Pivot Tables: Tạo nhiều PT cho từng phần Dashboard (trên sheet ẩn)

Tạo Pivot Charts: Tạo chart từ mỗi PT, chọn loại chart phù hợp (Chương 7)

Thiết kế layout: Tạo sheet "Dashboard" riêng, sắp xếp charts + KPIs

Thêm Slicer + Timeline: Kết nối tất cả PT với cùng Slicer/Timeline

Hoàn thiện: Ẩn gridlines, ẩn sheet data, protect sheet, format cuối cùng

Tạo KPI Cards

📊 KPI Cards bằng công thức GETPIVOTDATA

Tạo ô KPI bằng công thức lấy dữ liệu từ Pivot Table:

=GETPIVOTDATA("ThanhTien", 'PT_Data'!A3) → Tổng doanh thu =GETPIVOTDATA("MaDH", 'PT_Count'!A3) → Số đơn hàng =GETPIVOTDATA("ThanhTien",'PT_Data'!A3) / GETPIVOTDATA("MaDH",'PT_Count'!A3) → TB/đơn

Format các ô KPI: font size lớn (28pt), màu nổi bật, viền đẹp mắt.

🎨 10.4 Hoàn thiện Dashboard

✅ Checklist hoàn thiện

  • ☐ Ẩn gridlines: View → bỏ tick Gridlines
  • ☐ Ẩn row/column headers: View → bỏ tick Headings
  • ☐ Đặt background: Tô màu nền cho toàn bộ sheet (chọn màu tối hoặc xám nhạt)
  • ☐ Ẩn sheet dữ liệu: Click phải tab sheet → Hide
  • ☐ Protect sheet: Review → Protect Sheet (chỉ cho phép sử dụng Slicer)
  • ☐ Ẩn Field Buttons: PivotChart Analyze → Hide All
  • ☐ Remove scroll bars: File → Options → Advanced → bỏ tick scroll bars
  • ☐ Freeze panes: Nếu Dashboard dài hơn 1 màn hình → View → Freeze Panes
💡
Pro tip: Dùng zoom 80-85% để toàn bộ Dashboard vừa 1 màn hình. Kết hợp với Page Layout view để kiểm tra in ấn.

📊 10.5 Các mẫu Dashboard khác

👥 HR Dashboard (Nhân sự)

  • KPIs: Tổng nhân viên, Tỷ lệ nghỉ việc, Lương trung bình
  • Charts: Pie: Giới tính | Bar: Phòng ban | Line: Biến động nhân sự theo tháng
  • Slicer: Phòng ban, Chức vụ, Giới tính

📦 Inventory Dashboard (Kho)

  • KPIs: Tổng tồn kho, Giá trị tồn kho, Số SKU hết hàng
  • Charts: Bar: Tồn kho theo nhóm | Line: Xu hướng nhập/xuất | Table: Hàng sắp hết
  • Slicer: Nhóm hàng, Nhà cung cấp, Kho

💰 Finance Dashboard (Tài chính)

  • KPIs: Doanh thu, Chi phí, Lợi nhuận, Biên lợi nhuận %
  • Charts: Combo: Doanh thu (cột) + Biên LN (đường) | Waterfall: Phân tích chi phí
  • Slicer: Danh mục chi phí, Dự án, Quý

🎯 10.6 Bài tập thực hành

📝 Bài tập 10.1 – Xây dựng Sales Dashboard hoàn chỉnh

Đây là bài tập tổng hợp của toàn bộ giáo trình!

Yêu cầu:

  1. Dữ liệu: Tạo bảng bán hàng 100+ đơn hàng (MaDH, Ngay, KhuVuc, NhanVien, SanPham, NhomSP, SoLuong, DonGia, ThanhTien) — trong 6 tháng, 3 khu vực, 10 sản phẩm, 6 nhân viên
  2. Chuyển thành Table → Insert → PivotTable
  3. Tạo 5 Pivot Tables trên sheet riêng (ẩn sau):
    PT1: Doanh thu theo tháng | PT2: Doanh thu theo sản phẩm | PT3: Doanh thu theo khu vực | PT4: Top 5 NV | PT5: Doanh thu theo NhomSP
  4. Tạo Pivot Charts: PT1→Line | PT2→Bar | PT3→Doughnut | PT5→Column
  5. Sheet Dashboard: Sắp xếp charts + KPI cards theo layout mẫu
  6. Thêm Slicer (KhuVuc, NhomSP) + Timeline (Ngay) → kết nối tất cả PT
  7. Hoàn thiện: Ẩn gridlines, ẩn sheet data, format đẹp mắt

✅ Hướng dẫn tổng quát:

Bài này kết hợp kiến thức từ tất cả các chương:

  • Chương 1: Chuẩn bị dữ liệu đúng cách, chuyển thành Table
  • Chương 2-3: Tạo Pivot Tables, kéo thả fields
  • Chương 4: Định dạng PT với styles phù hợp
  • Chương 5: Tạo Calculated Field cho KPIs (LoiNhuan, TyLe...)
  • Chương 6: Thêm Slicer + Timeline, kết nối Report Connections
  • Chương 7: Tạo Pivot Charts, tùy chỉnh, ẩn Field Buttons
💡
Kết nối Slicer: Click phải → Report Connections → tick TẤT CẢ Pivot Tables. Khi click 1 nút trên Slicer → tất cả charts thay đổi cùng lúc!
