Tính toán nâng cao

Khám phá sức mạnh tính toán của Pivot Table: Value Field Settings, Show Values As, Calculated Field và Calculated Item.

📊 5.1 Value Field Settings

Mặc định, Pivot Table dùng Sum cho số và Count cho text. Bạn có thể thay đổi:

Click chuột phải vào ô Values → Value Field Settings...

Chọn hàm tính toán mong muốn trong tab Summarize Values By

📋 Các hàm tổng hợp (Summarize Values By)

Mẹo: Bạn có thể kéo cùng 1 field nhiều lần vào Values, mỗi lần chọn hàm khác nhau. Ví dụ: Sum of ThanhTien + Average of ThanhTien + Count of ThanhTien cùng lúc.

🔧 5.3 Calculated Field

Calculated Field cho phép tạo field mới bằng công thức từ các field hiện có, trực tiếp trong Pivot Table.

Click vào Pivot Table → tab PivotTable Analyze

Click Fields, Items & Sets → Calculated Field...

Đặt tên field (ví dụ: LoiNhuan)

Nhập công thức: = ThanhTien - GiaVon

Nhấn Add → OK

📊 Ví dụ thực tế: Lợi nhuận biên

Giả sử bạn có thêm cột GiaVon trong dữ liệu nguồn:

Calculated FieldCông thứcÝ nghĩa
LoiNhuan= ThanhTien - GiaVonLợi nhuận gộp
TyLeLN= LoiNhuan / ThanhTienTỷ lệ lợi nhuận %
LoiNhuanMoiSP= LoiNhuan / SoLuongLợi nhuận trên mỗi sản phẩm
⚠️
Hạn chế: Calculated Field tính trên tổng các field, không phải từng dòng. Ví dụ: = ThanhTien / SoLuong sẽ là Sum(ThanhTien) / Sum(SoLuong), có thể không chính xác. Dùng Power Pivot (Chương 8) nếu cần tính toán phức tạp hơn.

🎯 5.4 Bài tập thực hành

📝 Bài tập 5.1 – Phân tích tỷ lệ doanh thu

Yêu cầu:

  1. Tạo Pivot Table: Rows = SanPham, Values = Sum of ThanhTien
  2. Thêm ThanhTien vào Values lần 2 → đổi thành % of Grand Total
  3. Thêm ThanhTien vào Values lần 3 → đổi thành Rank Largest to Smallest
  4. Format: giá trị tiền #,##0, % là 0.0%
  5. Sản phẩm nào chiếm tỷ lệ doanh thu cao nhất?

✅ Kết quả mong đợi:

SanPhamDoanh thuTỷ lệHạng
Điện thoại64,000,00043.1%1
Laptop45,000,00030.3%2
Tablet30,000,00020.2%3
Tai nghe5,000,0003.4%4
Chuột4,500,0003.0%5

Kết luận: Điện thoại chiếm 43.1% doanh thu, xếp hạng #1.

📝 Bài tập 5.2 – Calculated Field: Lợi nhuận

Yêu cầu:

  1. Thêm cột GiaVon vào bảng dữ liệu (= SoLuong × đơn giá vốn)
  2. Tạo Calculated Field: LoiNhuan = ThanhTien - GiaVon
  3. Tạo Calculated Field: TyLeLoiNhuan = LoiNhuan / ThanhTien
  4. Hiển thị cả 3: Sum of ThanhTien, Sum of LoiNhuan, TyLeLoiNhuan
  5. Sản phẩm nào có tỷ lệ lợi nhuận cao nhất?

✅ Hướng dẫn:

Thêm cột GiaVon vào Table nguồn (ước tính 60-80% ThanhTien)

Refresh Pivot Table (Alt + F5)

PivotTable Analyze → Fields, Items & Sets → Calculated Field

Tên: LoiNhuan, Công thức: = ThanhTien - GiaVon → Add

Tên: TyLeLoiNhuan, Công thức: = LoiNhuan / ThanhTien → Add → OK

Format TyLeLoiNhuan: Number Format → Percentage → 1 decimal
